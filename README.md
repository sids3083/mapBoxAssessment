# 🏃‍♀️ Running

-   Clone repo Run `git clone https://github.com/sids3083/mapBoxAssessment.git`
-   Run `cd server && npm i`
-   Run `npm start`
-   See `http://localhost:8000`

    `Open Another Terminal`
-   Run `cd web && npm i`
-   Run `npm run dev`
-   See `http://localhost:3000`
